# Masquerade

**Masquerade（假面舞会视觉小说）** 是一款基于 Unreal Engine 5 开发的视觉小说，也是 Global Game Jam 2026 参赛作品。

游戏以“假面舞会”为核心意象，围绕身份、关系与选择展开叙事。玩家将在对话中推进剧情，通过角色立绘、CG、音乐、音效与 UI 交互体验故事氛围，并在关键节点作出选择。
